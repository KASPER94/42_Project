# Resources

Reference material for the project.

## From the subject

- **OSDev wiki (III.5)** — the documentation the author recommends as one of the
  best. There is a lot of documentation out there, good and bad; OSDev is the
  go-to. <https://wiki.osdev.org/>
- **GNU `ld` linker docs (IV.0.1)** — for creating your own linker file, the
  subject points to docs via a "here" link. The href is embedded in the PDF and
  not shown inline; the canonical reference is the GNU `ld` manual / OSDev
  "Linker Scripts" page.

## Useful OSDev starting points (not in the subject, commonly used for KFS_1)

- "Bare Bones" — minimal multiboot kernel walkthrough.
- "Multiboot" / Multiboot specification — the header GRUB expects.
- "Linker Scripts" — how to write the `.ld` file.
- "Printing to Screen" / VGA text mode — the 0xB8000 framebuffer used to display
  characters.
