# Bonus Part

The bonus part (chapter V).

You have a stable system? Nice. Now let's have some fun! Install whatever you
want — any software, GUI, ANYTHING. Make this system yours, with your touch.

Special points for:

- An X Server
- Window managers / desktop environments, like GNOME / LXDE / KDE / i3 / dwm …

> ⚠ **Warning:** The bonus part will only be assessed if the mandatory part is
> PERFECT. Perfect means the mandatory part has been integrally done and works
> without malfunctioning. If you have not passed ALL the mandatory
> requirements, your bonus part will not be evaluated at all.
