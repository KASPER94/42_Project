# ft_linux — how_to_train_your_kernel

**Version:** 3.6

**Summary:** Make your own linux distribution.

## What is this project?

ft_linux is a project where you build a basic, but functional, Linux
distribution from scratch. You build a Linux kernel, install a set of
binaries, lay out a filesystem hierarchy that complies with the standards, and
get the system connected to the Internet.

## Introduction

> Welcome to ft_linux. In this subject, you have to build a basic, but
> functional, linux distribution.
>
> This subject is not about Kernel programming, but it's highly related.
> This distro will be the base for all your kernel projects, because all your
> kernel-code will be executed here, on your distro.
> Try to implement what you want/need to. This is your userspace, take care of
> it!

## Purpose

This distribution becomes the base for all your subsequent kernel projects:
every piece of kernel code you write later will run here, on the distro you
built. The goal is a system that is genuinely yours — your userspace, your
configuration, your responsibility.
