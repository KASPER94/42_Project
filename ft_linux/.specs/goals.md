# Goals

The concrete objectives of ft_linux:

- Build a Linux Kernel.
- Install some binaries (see the [packages](packages.md) list).
- Implement a filesystem hierarchy compliant with the standards (this typically
  points to the FHS / Filesystem Hierarchy Standard — the anchor text in the
  source is "standards", with the target URL embedded in the PDF and not shown
  inline). See [resources](resources.md).
- Connect to the Internet.
